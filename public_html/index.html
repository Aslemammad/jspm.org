jspm 2.0-beta

ES module package manager

jspm is a JavaScript package manager designed to
work seamlessly with native modules in universal workflows,
while providing full compatlbility with the npm ecosystem.


  • Install any npm package with CommonJS automatically converted into ES modules.
  • Generate import maps to load any package in the browser from jspm_packages without a build step.
  • Easily build and optimize packages.
  • Integrate with third-party tools with just a resolver hook.

Read more about jspm and its architecture, or jump straight in with the guide.
