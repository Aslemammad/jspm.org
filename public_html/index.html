jspm 2.0-beta.1

First public experimental beta, 25 April 2019

jspm is a JavaScript package manager designed to
work seamlessly with native modules in universal workflows,
while providing full compatlbility with the npm ecosystem.


  • Install any npm package into flat, symlinked jspm_packages.
  • Generate an import map to load any package in the browser without a build step.
  • Easily optimize packages for production.
  • Integrate with third-party tools with just a resolve hook.

Read more about the features and architecture, or jump straight in with the getting started guide.
